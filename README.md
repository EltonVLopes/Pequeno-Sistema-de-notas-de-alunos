📌 Sistema de Cadastro de Alunos e Notas (Python)

Este projeto é um sistema simples desenvolvido em Python para praticar lógica de programação e estruturas de dados básicas. Ele permite cadastrar um aluno, registrar suas notas, calcular a média e informar se ele foi aprovado, está em recuperação ou reprovado.

⚙️ O que o sistema faz
Solicita o nome do aluno
Recebe duas notas
Calcula a média final
Armazena os dados do aluno em uma estrutura de dados
Exibe a média e o status do aluno:
Aprovado (média ≥ 7)
Recuperação (média ≥ 5 e < 7)
Reprovado (média < 5)
Exibe os dados armazenados em uma lista
🧠 Conceitos e métodos aprendidos

Durante o desenvolvimento deste projeto, foram aplicados os seguintes conceitos:

Entrada de dados com input()
Conversão de tipos (float)
Estruturas condicionais (if, elif, else)
Estruturas de dados:
Lista (list) para armazenar alunos
Dicionário (dict) para organizar dados de cada aluno
Manipulação de listas com .append()
Operações matemáticas básicas (cálculo de média)
Formatação de saída com f-strings
Organização básica de lógica de sistema
📚 Objetivo do projeto

Este projeto tem como objetivo praticar fundamentos da linguagem Python e entender como construir pequenos sistemas utilizando estruturas de dados simples, simulando um sistema básico de gerenciamento de alunos.
